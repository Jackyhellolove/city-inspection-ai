# 城市巡查AI助手

## 项目介绍

利用无人机、人工智能技术，
辅助城市设施巡查和运维管理。

## 主要功能

- 无人机巡查数据管理
- AI识别城市问题
- 自动生成巡查报告
- 问题闭环管理

## 应用场景

- 城市道路
- 园林环卫
- 市政设施
- 城市生命线

## 开发计划

第一阶段：
完成项目框架

第二阶段：
接入AI识别

第三阶段：
开发管理平台

## 运行方法

### 桌面界面

在项目文件夹的终端中运行：

```bash
python3 app.py
```

输入巡查问题并点击“开始识别”，即可查看问题类型、处理部门和优先级。

如果 Mac 上的桌面控件显示不正常，请使用更稳定的浏览器版：

```bash
python3 web_app.py
```

浏览器版支持自动生成问题编号，记录发现地点、负责人和处置状态，并可保存历史记录和导出 CSV 表格。记录仅保存在当前浏览器中。

### 命令行版本

```bash
python3 inspection.py
```
